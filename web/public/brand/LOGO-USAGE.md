# VIDRA Logo Pack

## Files
- `vidra-logo-primary.svg`: use on dark backgrounds.
- `vidra-logo-light-bg.svg`: use on light backgrounds.
- `vidra-mark.svg`: icon mark only (app icon/social avatar).
- `vidra-favicon.svg`: favicon source.

## Quick usage
In Next.js:

```tsx
import Image from "next/image";

<Image src="/brand/vidra-logo-primary.svg" alt="Vidra by Lexa AI" width={320} height={86} priority />
```

## Color references
- Deep background: `#0E1526`
- Blue accent: `#7EA7FF`
- Mint accent: `#70E0D2`
- Light text: `#EEF4FF`
